Create an application:
An AI health chatbot that uses gemini 2.5 flash model. 
You can chat to it about your health, tell it symptoms and based on that it will diagnose you. 
If the conditions seem critical and not managable / self diagnosable, it should, on the UI of the chat, show nearby hospitals or doctors that you can contact for related conditions.

Tech stack: 
- React + Shadcn frontend
- FastAPI backend
- mongodb database
- Docker for build
